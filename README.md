## 📁 Como baixar o projeto

```bash

# Clonar o repositório
$ git clone https://github.com/Dani2003G/Projeto_clone_spotify

# Entrar no diretório
$ cd Projeto_clone_spotify

# Instalar as dependências
$ yarn install

# Iniciar o projeto
$ yarn start

```
